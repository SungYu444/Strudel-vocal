# My Strudel Songs

Song created with Strudel, the live coding music tool

## How to Play

1. GO TO: **https://strudel.cc**
2. Copy the code from any `.strudel` file
3. Paste it into the editor
4. Press Play ▶️ or hit `Ctrl+Enter`

## Songs

### Dreamy Guitar
Soft, atmospheric piece with gentle nylon guitar and lush reverb in C minor.

### Blurry Vibes
Ethereal soundscape with heavy delay, reverb, and floating melodies.
